# Forge Digital Technologies Landing Page

## Project Overview
This project is a React-based single-page landing site for a digital agency named "Forge Digital Technologies", built precisely to match the provided high-fidelity design references. It uses React 18, Tailwind CSS, and Lucide icons.

## Features & Sections
1. **Header**: Fixed navbar with blur effect.
2. **Hero Section**: Tagline, descriptions, call to action buttons, and stats grid.
3. **Services**: Grid view of tech services offered.
4. **About**: Brief overview of the team's mission and core values.
5. **Process**: 5-step horizontal timeline component.
6. **Portfolio**: Card layout for recent works.
7. **Features**: "Why Choose Us" grid.
8. **CTA / Footer**: Final call to action section followed by comprehensive footer links.

## Design Assets & Configuration
- **Color Theme**: Deep navy/black (`#06060c`), accents of vibrant purple and blue.
- **Typography**: Uses 'Inter' via Google Fonts.
- **Icons**: Lucide static font (`icon-name`).
